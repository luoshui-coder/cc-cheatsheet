# Claude Code Cheat Sheet

Static single-page cheat sheet, deployed via GitHub Pages.

## Local preview
Open `index.html` directly in a browser.

## Deploy
Push to `main`; GitHub Actions publishes the site.

## Weekly refresh
A weekly scheduled GitHub Actions run is configured, but the current page is a static HTML snapshot and does not yet contain any fetch/update script. Add a data-generation step if you want scheduled auto-refresh from official Claude sources.
